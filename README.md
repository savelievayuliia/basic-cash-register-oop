# basic-cash-register-oop
# A model of a simple retail cash register. 

## BRIEF

* RetailItem_class.py - contains of a RetailItem class set/get methods for 3 item's parameters: description, quantity and price
* CashRegister.py - contains of methods of interaction with RetailItem objects: adding objects to the cart, showing "Total to checkout message" and clearing the shoping cart
* both .py files have to be downloaded to the same python package
